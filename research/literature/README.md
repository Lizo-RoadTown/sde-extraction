# literature/

Papers, lit reviews, annotated bibliographies.

## Convention

- One file per paper or topic: `<author-year-short>.md` or `<topic>-lit-review.md`
- Include: citation, source URL or DOI, key claims, methods, limitations, your notes
- For full PDFs: store in `data/` (gitignored) if proprietary; otherwise external link only

Lean on `skills/document-parsing` to extract structure from sources efficiently.
