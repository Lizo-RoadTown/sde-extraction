When the system finishes reading a paper, it replays the search for you. A magnifying glass moves
across the real page image and lands on the exact place each value was found — so you can see, not
just be told, where every number came from.

## What you see

- the actual page from the paper, dimmed
- the lens glides to each value and highlights the exact spot it was found
- the value appears in the list beside the page as the lens lands on it

_(expand: a labelled screenshot of the lens mid-search.)_

## Why you can trust it

The lens only lands where the value's **exact quote** was found in the paper's own text. A highlight
means the words are really there, on that page, as written — the position is read from the paper, not
guessed. This is the same source-of-truth idea as [Where each value comes from](/explanation/provenance/).

_(expand: verbatim quote → real position → why that is a real check, not decoration.)_

## When it can't pinpoint a value

Some values can't be located on the page — they live in a scanned table or a figure with no text
behind it, or the wording differs slightly. The system does **not** invent a highlight for those.
They are shown honestly as **“· no position”** in the list, and the lens stays put. A value with no
highlight was still captured; it just couldn't be pinned to a spot.
