## What this documents

This is the reference record for a method that extracts **stochastic differential equation
(SDE) epidemiological models** from academic papers into a **provable, structured form** —
where every captured value is either present (transcribed verbatim from the paper, with its
source) or explicitly absent, and every data hand-off is observable and provenance-tracked.

It is written to a single standard: **rigour**. Nothing here is asserted that is not
substantiated by the system itself. Where a capability is planned but not yet verified, the
page says so plainly rather than describing something that does not yet exist.
