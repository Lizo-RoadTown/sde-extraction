title: How to read this
description: What this documentation is for and where to start.
---

This documentation is for scientists who work with stochastic epidemiological models. It does not
assume you know programming or AI. It explains what the system reads out of a paper and how you can
check it against the source.

There are three things you will want to do:

1. Follow a paper from figure to model. Start with [Follow a paper, step by step](/start/reproduce/).
2. Check a value yourself. See [Where each value comes from](/explanation/provenance/).
3. Watch the process and see how sure it is. See [Watch it work](/explanation/observability/).

If you only read one page, read [Follow a paper, step by step](/start/reproduce/).
