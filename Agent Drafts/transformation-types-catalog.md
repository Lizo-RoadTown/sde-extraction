# Transformation types in lifting a stochastic (SDE) epidemiological model from a paper

The point of this list: enumerate every transformation an LLM could hit between *what a paper shows*
and *the executable model*, and tag each as **DETERMINISTIC** (a fixed rule a script can apply +
verify — i.e. workflow) or **JUDGMENT** (the figure/symbols are genuinely ambiguous; a real decision
must be made and recorded with provenance — i.e. an autonomous-decision/HITL seam).

**The single most important finding:** for almost every transformation, the *algebra is deterministic*
once the inputs are fixed — the *judgment* is *detecting which form/convention the paper used*. So the
design isn't "workflow vs autonomous"; it's "deterministic apply, wrapped around a judgment-flagged
*detect*." That maps straight onto the determinism-web principle.

Mapped to the step sequence: **Step 1** = lift the figure. **Step 2** = find the stochastic model
(drift + diffusion) that matches *that* figure → Families A (bridge from figure) + B + C + E. **Step 3**
(only after 1 & 2) = the parameters → Family D. Family F (numerical) is for the reproduction oracle.

---

## Family A — Observable transforms: what the FIGURE plots vs the model's state (the Step-1→2 bridge)

A figure rarely plots a raw state variable. To know which model matches it, you must know what the
curve *is*.

| # | Transformation | What's plotted vs the state | Verdict |
|---|---|---|---|
| A1 | Prevalence | `I` (or `I/N`) — the raw state (the null case) | DETERMINISTIC |
| A2 | Incidence (new cases/time) | the **flux** `βSI/N`, a flow — NOT the `I` stock | **JUDGMENT** (label "cases" is stock/flow ambiguous) |
| A3 | Force of infection λ(t) | `βI/N` (per-susceptible hazard) | DETERMINISTIC (if labeled) |
| A4 | Cumulative incidence / cases | `∫ incidence` = `N − S` (monotone) | DETERMINISTIC |
| A5 | Attack rate / final size | `1 − S∞/S₀` (asymptotic scalar) | DETERMINISTIC |
| A6 | Mortality / deaths | `∫ μI` via a D compartment (lagged) | DETERMINISTIC (JUDGMENT if IFR-scaled/lagged) |
| A7 | Hospitalizations | derived H: occupancy (stock) vs admissions (flow) | **JUDGMENT** |
| A8 | R₀ | scalar `β/γ` or spectral radius of next-gen matrix | **JUDGMENT** (which NGM decomposition) |
| A9 | Effective Rₜ | `R₀·S/N` or an external estimator | **JUDGMENT** (provenance/estimator) |
| A10 | Phase plane (state vs state) | time is implicit; axes are two states | **JUDGMENT** (dynamical-systems reading) |
| A11 | Bifurcation / outcome-vs-parameter | x = parameter, y = equilibrium/final size (not time) | DETERMINISTIC (once x-axis read) |
| A12 | Spatial/network snapshot | one time slice over space — no time axis | DETERMINISTIC (JUDGMENT if axes unlabeled) |

**Aggregation (the curve is a SUM of compartments)** — detecting it is JUDGMENT; the sum is mechanical:
A13 multi-stage Erlang `I=ΣIᵢ` · A14 staged/differential-infectivity (HIV) · A15 age/risk-group sums
(WAIFW) · A16 spatial/patch sums · A17 host vs vector totals · A18 clinical-status sums
(symptomatic+asymptomatic). All **JUDGMENT** to detect.

**Stochastic-specific plotted quantities** — what statistic is the curve? A19 single sample path
(jagged/step) · A20 ensemble mean (NOT the ODE — provably below it) · A21 mean±SD band · A22 quantile
fan · A23 prediction vs confidence/credible interval · A24 time-dependent distribution (bimodal) ·
A25 quasi-stationary distribution histogram · A26 final-size distribution · A27 extinction-time
(expected, and distribution) · A28 outbreak-vs-extinction probability `(1/R₀)^{i₀}`. The histograms
(A24–A27) are DETERMINISTIC to classify once axes are read; **distinguishing path vs mean vs ±SD vs
quantile vs CI (A19–A23) is the most pervasive JUDGMENT call** — they all look alike and captions
under-specify.

## Family B — Mathematical-form transforms of the SDE itself (Step 2)

The same model written/derived in a different but equivalent (or approximating) form.

| # | Transformation | What it changes (the rule) | Verdict |
|---|---|---|---|
| B1 | Itô ↔ Stratonovich | drift `+½ b b′`; diffusion unchanged | apply DET · **detect interpretation JUDGMENT** |
| B2 | Itô's lemma (general change of var) | drift `+½ b² f″` | DETERMINISTIC |
| B3 | Lamperti transform (→ unit diffusion) | `Y=∫dx/b`, drift `a/b − ½b′` | apply DET · recognize/multivariate JUDGMENT |
| B4 | Log / GBM | `−½σ²` drift shift | apply DET · which rate reported JUDGMENT |
| B5 | General `f(X)` (logit of prevalence, etc.) | `½b²f″` correction | apply DET · recognize JUDGMENT |
| B6 | Non-dimensionalization / rescaling | dimensionless groups; noise must rescale | algebra DET · noise scaling JUDGMENT |
| B7 | Scale to N (counts ↔ fractions) | diffusion `× 1/√N` | given N DET · detect convention JUDGMENT |
| B8 | Time-change / rescaling | drift `×1/c`, diffusion `×1/√c` (asymmetric) | DETERMINISTIC (auto-check the powers) |
| B9 | Kramers–Moyal (truncate at 2) | master eq → drift α₁, diffusion √α₂ | apply DET · validity JUDGMENT |
| B10 | Chemical Langevin (CLE, from reactions) | per-reaction `√(aⱼ)` noise; `BBᵀ=Σ νⱼνⱼᵀ aⱼ` | from reaction list DET · from prose JUDGMENT |
| B11 | Kurtz diffusion limit | drift = the ODE; noise `O(1/√N)` | scaling DET · regime JUDGMENT |
| B12 | van Kampen Ω / Linear Noise Approx (LNA) | linear OU; diffusion frozen on `φ(t)` | **JUDGMENT** (classify CLE vs LNA) |
| B13 | SDE ↔ Fokker–Planck | `b=√(2D)`; factorization non-unique | SDE→FPE DET · FPE→SDE factorization JUDGMENT |
| B14 | Moment / cumulant equations | `+E[b²]` variance source; needs closure | **JUDGMENT** (no closure for nonlinear) |
| B15 | Deterministic limit (drop noise → ODE) | `b→0`, drift → ODE | DET + interpretation caveat |
| B16 | Linearization around equilibrium (OU) | Jacobian drift, frozen diffusion | apply DET · scope JUDGMENT |
| B17 | Compartment aggregation / lumping | sum rows of B; **carry covariances** | **JUDGMENT** |
| B18 | Quasi-steady-state reduction | adds a *stochastic* correction term | **JUDGMENT (high)** |

> The recurring failure mode across B is the **diffusion term, not the drift** — the `½bb′`, `1/√N`,
> `1/√c`, per-channel √-structure, frozen-on-trajectory diffusion, and aggregation covariances all live
> in `b`/`B`. Treat diffusion as the high-risk field.

## Family C — Notation / representation transforms (Step 2)

The "same quantity written differently" problems.

| # | Transformation | Verdict |
|---|---|---|
| C1 | Symbol → variable/parameter mapping (subscripts, Greek, primes, hats; `β_vh` ≠ `β_hv`) | **JUDGMENT** |
| C2 | Implicit multiplication / juxtaposition / precedence (`βSI/N`, `β_vh` atomic) | parse JUDGMENT → then DET |
| C3 | Σ / index expansion; vector/matrix ↔ scalar | DETERMINISTIC (given index-set size) |
| C4 | Seasonal / time-dependent forcing `β(t)=β₀(1+β₁cos…)`; steps; delays | formula DET · **units/phase/amplitude JUDGMENT** |
| C5 | Differential notation (dX, Ẋ, dX/dt) → Euler–Maruyama increment (`√Δt` on noise!) | DETERMINISTIC |
| C6 | Units / dimensional consistency check | DETERMINISTIC (a checker script) |

## Family D — Parameter transforms (Step 3 — only after the model is fixed)

| # | Transformation | Rule | Verdict |
|---|---|---|---|
| D1 | Rate ↔ probability | `p = 1 − e^{−rΔt}` | DET (matrix-exp branch under competing risks) |
| D2 | Frequency- vs density-dependent (`βSI/N` vs `βSI`) | sets β's units | **JUDGMENT** to decide · DET to convert |
| D3 | R₀ ↔ β; back-calc unstated param | `β = R₀γ` (SIR), NGM in general | algebra DET · **which R₀ definition JUDGMENT** |
| D4 | Mean/duration ↔ rate | `γ = 1/D` | DETERMINISTIC (non-exponential stage = minor JUDGMENT) |
| D5 | Nondimensional ↔ dimensional; scale by N | `S = s·N`; undo `τ=γt` | DET (given N + convention) |
| D6 | Per-time rescaling (day/week/year) | rates scale linearly; probs do not | DETERMINISTIC |
| D7 | Fitted vs assumed vs cited; range vs point | which value made *this* figure | **JUDGMENT** |
| D8 | Compound-unit conversion (`β = c·q`) | track each factor | DET (given units) · product-vs-factor JUDGMENT |

## Family E — Noise-structure transforms (Step 2; load-bearing for reproduction)

| # | Transformation | Verdict |
|---|---|---|
| E1 | Additive vs multiplicative (state-dependent) diffusion | DET if written · JUDGMENT if verbal |
| E2 | Demographic (`√rate`, from CLE) vs environmental (`β→β+σdW`, white/OU) | DET if equation given · JUDGMENT to rebuild |
| E3 | Driver: Wiener vs Poisson/Lévy jumps vs Markov-switching/telegraph | read type DET · **engine coverage JUDGMENT** |
| E4 | Noise-term count & correlation; `B = V^{1/2}` (matrix sqrt, NOT elementwise/diagonal) | DET if G/B given · factorization JUDGMENT |
| E5 | Noise placement & conservation (one shared dW, opposite signs, per flow) | DETERMINISTIC (mechanical constraint + guard) |

| # | Transformation | Verdict |
|---|---|---|
| F1 | Scheme & order (Euler–Maruyama vs Milstein; additive ⇒ EM = Milstein) | read DET · choose-when-unstated JUDGMENT |

## The decision this list is for: workflow vs autonomous

A clean pattern falls out:

- **The math is always a workflow.** Once the form/convention/value is known, every transformation
  here is a fixed rule a script applies and can verify (units, term counts, conservation, fixed-seed
  re-simulation). These are the determinism-web seams.
- **The genuine autonomous decisions are the *detections*:** which interpretation (B1), which R₀
  definition (D3), frequency vs density (D2), is the curve a path or a mean (A19–A23), did aggregation
  happen (A13–A18), which value made this figure (D7), what dt/seed/positivity-fix when unstated
  (F3–F5). These are where a real choice is made and must be recorded with provenance (the HITL track).

So the open question for us isn't "workflow or autonomous" globally — it's **per transformation, is the
*detection* reliable enough to automate-with-a-check, or does it need a human gate?** That's the next
conversation, Step 2, not yet decided.
